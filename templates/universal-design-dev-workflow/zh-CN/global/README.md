# Global conventions / 全局规约

### English

This folder holds **cross-language norms** for this kit: artifact names, directory parity, SSOT rules, and adoption layout. **Normative detail:** [`ARTIFACT-LAYOUT.md`](./ARTIFACT-LAYOUT.md), [`SSOT-AND-MAINTENANCE-RULES.md`](./SSOT-AND-MAINTENANCE-RULES.md). The same `global/` tree is duplicated under `en/global/` and `zh-CN/global/` so each locale root is self-contained. If a rule conflicts with another file in the same locale tree, **update `global/` first**, then align the rest.

### 简体中文

本目录存放本 kit 的**跨语言全局规约**：制品命名、目录镜像、单一事实来源（SSOT）规则与落地方式。**细则见** [`ARTIFACT-LAYOUT.md`](./ARTIFACT-LAYOUT.md)、[`SSOT-AND-MAINTENANCE-RULES.md`](./SSOT-AND-MAINTENANCE-RULES.md)。`global/` 在 `en/global/` 与 `zh-CN/global/` **各有一份副本**，以便每个语言根目录自洽。若与同树其他文件冲突，**先改 `global/`**，再同步其余文档。
