# Universal design-to-development workflow kit / 通用设计—研发工作流套件

### English

**Start:** [`ENTRY.md`](./ENTRY.md) — bilingual guide, repository layout, and **copy-paste bootstrap prompts** for the full workflow.

| Path | Role |
|------|------|
| [`ENTRY.md`](./ENTRY.md) | Single entry / 唯一入口 |
| [`en/`](./en/) | Locale tree **A** (mirror of `zh-CN/`) / 语言树 A（与 zh-CN 镜像） |
| [`zh-CN/`](./zh-CN/) | Locale tree **B** / 语言树 B |
| [`en/global/`](./en/global/) | Global norms **copy 1** / 全局规约副本一 |
| [`zh-CN/global/`](./zh-CN/global/) | Global norms **copy 2** / 全局规约副本二 |

Each locale tree is self-contained (`global/` inside it). **Every Markdown file in this kit is written in both English and 简体中文** in the same file (or clearly paired sections). Typically `en/` and `zh-CN/` files are kept **identical** so you can copy either tree into `docs/`.

### 简体中文

**入口：** [`ENTRY.md`](./ENTRY.md) — 中英说明、目录说明、**一键复制主提示词**。

| 路径 | 用途 |
|------|------|
